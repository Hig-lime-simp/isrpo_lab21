## Описание (что изучили)

В ходе выполнения лабораторной работы я создал полноценное веб-приложение "TODO-лист" (список задач), которое позволяет:

- Добавлять новые задачи
- Отмечать задачи как выполненные
- Возобновлять задачи
- Удалять задачи
- Отслеживать общее количество задач

---

### 1. Работа с DOM (Document Object Model)
- Поиск элементов с помощью `getElementById()`
- Создание новых элементов через `createElement()`
- Добавление элементов в DOM с помощью `appendChild()`
- Удаление элементов через `remove()`
- Управление классами элементов с помощью `className` и `classList`

### 2. Обработка событий
- Назначение обработчиков через `addEventListener()`
- Обработка кликов на кнопках
- Обработка нажатия клавиш (Enter в поле ввода)
- Событие загрузки страницы `DOMContentLoaded`

### 3. Работа с формами и данными
- Получение значений из полей ввода через `.value`
- Валидация данных (проверка на пустую строку)
- Очистка полей после ввода
- Установка фокуса через `.focus()`

### 4. CSS-стилизация
- Применение классов для визуального оформления
- Стилизация состояний элементов (hover, focus)
- Адаптивный дизайн

## Демонстрация работы

### Основные функции приложения:

1. **Добавление задачи**
   - Ввод текста в поле
   - Нажатие кнопки "Добавить" или клавиши Enter
   - Проверка на пустой ввод

2. **Управление задачами**
   -  Кнопка "Выполнено" — зачёркивает задачу и меняет цвет
   -  Кнопка "Возобновить" — возвращает задачу в активное состояние
   -  Кнопка "Удалить" — полностью удаляет задачу

3. **Счётчик задач**
   - Автоматически обновляется при добавлении/удалении задач
   - Отображает общее количество задач в списке

4. **Примеры при загрузке**
   - При открытии страницы автоматически загружаются примеры задач
   - Демонстрирует работу приложения сразу после запуска
